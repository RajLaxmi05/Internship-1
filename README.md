# 🌦️ India Weather Dashboard

An interactive **Streamlit-based weather dashboard** that visualizes historical weather data across Indian cities.  
The dashboard allows users to explore temperature trends, precipitation patterns, and monthly weather statistics using interactive filters.

## 🔗 Live Dashboard
👉 **Click here to view the app:**  
https://rajlaxmi05-internship-1-app-qmqg6s.streamlit.app/

## 📊 Key Features
- City-wise and year-wise weather analysis
- Monthly average temperature & precipitation trends
- Interactive charts using Plotly
- Clean and responsive Streamlit UI

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly


## 🚀 How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
